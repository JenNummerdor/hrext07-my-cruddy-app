Intended Purpose

Each week, log your data. It will store in your local memory. Currently, the data is based on a subjective rubric, so answer the questions as best as you can. The questions that were chosen for this project are examples, and the rubrics to grade them are subjective. Feel free to edit them to your own benefit.

Language

This was written in jQuery with some Javascript elements. I wanted to use Bootstrap as well so that I didn't have to worry about CSS. I also used C3 which generates graphs of data. This visual is nice to compare data week-by-week. The goal is to get the user to begin to notice trends in the data to see how to see which health goals they are exceeding and which ones can be improved upon.
